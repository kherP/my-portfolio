Overview

Occupational pension portal is a portal that allows administrator of an organisation to manage their employees' insurances. This portal is only meant for certain users but not public.

Client

Leading Nordic corporate bank

My role

Lead frontend developer

Challenges

  • Sensitive information. The team will need to be extra mindful on data handling.
  • Huge dataset and different ways of using the system. We tried to use external virtual list library to improve the performance of the page with large list. However, each user has different way to use our system. Some of them prefer to use our site with some zoom settings. This causes the virtual list library logic breaks. In order to resolve this issue, we introduced our own virtual list hook.
  • Rapid feature delivery. Every two weeks, we introduce new features or bug fixes. Worrying that some users might miss out the new changes, we added in feature guide to notify users about the changes.
  • Pension insurance complexity The team overcomes the complexity by making sure the process for making changes to be clear and simplified.

The result

  • Old system partially replaced. We are in the midst of moving the remaining features to this digital system.
  • Significant increase in customer satisfaction level We improve the portal from time to time based on the feedback we received.
  • 17, 000 users target reached! We have reached 17, 000 users for this portal and the number is still increasing.
