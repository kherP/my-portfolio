Overview

A set of React components library based on brand styles guideline. The library provides high flexibility where users can easily customize the components to fit the design. You can find the documentation of the library here.

My role

Contributor, Reviewer

Challenges

  • Components' flexibility. The components used to have lots of limitations where users don't have much freedom to overwrite the style of the components. We refactored the components in a more declarative way.
  • Accessibility. The brand I'm working with is moving to inclusive design. Thus, we are also adding in the missing accessibility to the all the components. New version will be released soon.

The result

  • Easier implementation. The developers now have more flexibility when implementing this library in their projects. They can also define the components with default values.
  • Shorter development times. Reusable components help to reduce the development times drastically.
