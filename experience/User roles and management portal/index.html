Overview

This internal portal is interrelated to Occupational pension portal. System administrator can view and manage user roles for Occupational pension portal here.

Client

Leading Nordic corporate bank

My role

Lead frontend developer

Challenges

  • Microfrontend. This application is one of the microfrontends within a shell. We will need to pay extra attention to prevent style leakage. In order to standardize CSS styles throughout the shell and to prevent CSS styles leaks, we use SMACSS methodology in this project.
  • Users' access level. Some of the users are not allowed to view certain changes. Thus, it increased the complexity of the application. We introduced retricted components in Frontend where the visibility of the components/ data will be set based on the permissions returned from BFF. On API level, we also check whether the user is allowed to view the data requested.

The result

  • Microfrontend's style aligns with the shell. The microfrontend embedded nicely within the designated shell.
  • Data access control works well. Users with different access level can only access the data they are allowed to.
  • Improved user experience for Occupational Pension Portal. The customer service can now easily assist the users of Occupational Pension Portal to manage their users.
